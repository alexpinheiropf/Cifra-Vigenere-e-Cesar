=========== ALGORITMO DESENVOLVIDO PARA CRIPTAÇÃO DE MENSAGENS UTILIZANDO ===========
================ CIFRAS DE VIGENERE E CESAR JUNTO COM A TABELA ASCII ================




PARA EXECUÇÃO DO PROGRAMA DEVE-SE TROCAR A EXTENÇÃO DO ARQUIVO "Criptografia_vigenere_e_cesar.doc"

PARA "Criptografia_vigenere_e_cesar.jar".





O programa utiliza a cifra de vigenere para primeira codificação e a cifra de cesar junto com a tabela ASCII para segunda criptação.

****** PARA UTILIZAR O ALGORITMO DEVE SEGUI O SEGUINTE PASSO A PASSO ABAIXO *****

1. NO COMPUTADOR QUE FOR EXECUTADO DEVE-SE ESTAR INSTALADO O PROGRAMA DO JAVA PARA O FUNCIONAMENTO:
****** Pode se encontrar a versão mais atualizada no link abaixo ***********
		https://www.java.com/pt_BR/download/

2. APÓS INICIAR O PROGRAMA ESCOLHER SE DESEJA CODIFICAR OU DECODIFICAR ALGUMA MENSAGEM

3. EM SEGUENCIA DEVE-SE DIGITAR A "MENSAGEM E A CHAVE!!!" EM LETRA "MAIUSCULA!!!" PARA PODER CODIFICAR OU DECODIFICAR

4. POR FIM IRÁ APARECER A MENSAGEM COM AS DUAS CIFRAS UTILIZADAS.
